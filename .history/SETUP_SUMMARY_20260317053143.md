## 🆘 Troubleshooting

**Q: Images not showing after copy?**
- A: Restart frontend with `npm run dev`
- A: Check browser console for 404 errors

**Q: Which file is MoMo vs TPBank?**
- A: Check your image - MoMo has pink logo, TPBank has orange logo
- A: Save MoMo → `qr-momo.png`
- A: Save TPBank → `qr-tpbank.png`

**Q: Can't approve payments?**
- A: Check you're logged in as admin
- A: Go to `/admin/payment-approval` 
- A: Payments should appear in "Chờ xác nhận" tab
